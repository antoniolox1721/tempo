# ⏱️ Tempo — web app

Installable web app (PWA) for [Tempo], a natural-language reminders app:
type **"call the dentist tomorrow at 3pm"** and it becomes a scheduled reminder.

**https://antoniolox1721.github.io/tempo/**

On an iPhone: open in Safari → Share → **Add to Home Screen**.

This repository only hosts the built, minified bundle for deployment.
The application source is developed in a private repository.
